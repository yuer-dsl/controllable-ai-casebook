# controllable-ai-casebook
Authoritative casebook for the Controllable AI Acceptance Standard. This repository contains the first paired MAOK audit decisions (FAIL / ALLOW), demonstrating how AI-enhanced decisions are either prohibited or legally permitted to enter the real world under a charter-based governance framework.
This repository is the authoritative **casebook** for the
**Controllable AI Acceptance Standard**.

It contains the first paired MAOK audit decisions that demonstrate,
with full audit traces, how AI-enhanced decisions are:

- **Rejected (FAIL)** when they violate controllability requirements
- **Approved (ALLOW)** when they are structurally compliant and
  legally permitted to enter the real world

This repository does **not** contain code.
It contains **evidence**.

## Included Cases

- **MAOK-AT-0001 (FAIL)**  
  A traditional AI-driven asset allocation proposal,
  rejected due to structural non-controllability.

- **MAOK-AT-0002 (ALLOW)**  
  A human-owned, AI-enhanced asset allocation decision,
  legally permitted to enter the real world under MAOK charter review.

Together, these two cases form the **first complete negative/positive
evidence pair** for Controllable AI decision governance.
